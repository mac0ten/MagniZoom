# Security Policy

MagniZoom uses Android overlay, camera, and MediaProjection APIs. Please report issues involving unintended capture, permission bypasses, protected content capture, or data exposure responsibly.

## Reporting a Vulnerability

If you find a security issue, please avoid opening a public issue with exploit details. Contact the project maintainers privately instead.

Include:

- A concise description of the issue.
- Android version and device model.
- Reproduction steps.
- Any relevant logs, screenshots, or proof of concept details.

## Supported Versions

This prototype currently supports the latest `main` branch only.
